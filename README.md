# Homebrew Tap

This is the official Homebrew tap for tools maintained by [@sorokin-vladimir](https://github.com/sorokin-vladimir)

It contains Homebrew Formulae for CLI/TUI tools such as:

- `tele` — TUI Telegram client

---

## Installation

### Step 1 — add the tap

```bash
brew tap sorokin-vladimir/tap
```

### Step 2 — install a tool

```bash
brew install tele
```
